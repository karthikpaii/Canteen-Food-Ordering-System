# Canteen-Food-Ordering-System

A full-stack Canteen Management System developed with PHP and MySQL that allows users to browse menu items, place orders, and track order status, while admins can manage items, orders, and users through a dedicated dashboard.

### Features
- User Registration & Login
- Browse Menu Items
- Place & Cancel Orders
- Order Status Tracking
- Admin Dashboard
- Add/Edit/Delete Menu Items

###  Tech Stack
- PHP
- MySQL
- HTML, CSS
- JavaScript

### Upcoming Features

- Form Validation
- Payment Option
